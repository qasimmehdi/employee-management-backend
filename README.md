# employee-management-backend

## Biult on Deno

## Install deno

## Shell (Mac, Linux):

`curl -fsSL https://deno.land/x/install/install.sh | sh`

## PowerShell (Windows):

`iwr https://deno.land/x/install/install.ps1 -useb | iex`

## Run:

`deno run --allow-net --unstable --allow-read --allow-write --allow-plugin mod.ts`
